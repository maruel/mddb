# mddb

A lightweight Notion-like web application with markdown-based storage. Build and organize documents and databases entirely through a modern web interface, with all content persisted as markdown files.

## Features

- **Markdown-First Storage**: Pages and databases are stored as markdown files for simplicity and portability
- **Image Assets**: Images are stored directly in the file system with proper referencing
- **Rich Web UI**: Built with SolidJS for responsive, reactive interfaces
- **RESTful API**: Go-based backend providing clean API endpoints
- **Real-time Editing**: Fast, responsive editing experience
- **Database Tables**: Markdown-based database records with structured queries

## Tech Stack

- **Frontend**: SolidJS (TypeScript)
- **Backend**: Go 1.25+
- **Storage**: File system (markdown + images)
